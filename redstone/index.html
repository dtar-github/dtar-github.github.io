The Redstone Generator

The redstone generator generates redstone components that have certain properties. It is very useful when you're trying to figure out what component to use that needs to do something specific. The first checkbox indicates whether that property is filtering out the components. If it is off, it doesn't filter anything. The second checkbox determines whether the property is true of false. For example, if the first checkbox is on and the second one is off, it filters through the components that don't have that property. If the second one is on, it will filter the components that have that property. If the first checkbox is off, it won't filter out any components for that property.
